Q1: Why might we consider using Git?

Answer: Git is a useful version control system that allows for us to track our changes (version control) and facilitates collaboration because it can be uploaded to an online repository for others to look at.

Q2: What field could data science be applied to? Give an Example.

Answer: Data science can be applied to any field that involves any sort of data. For example, data science could be applied to cybersecurity, in which data scientists have to be able to quickly detect anomalies and threats through collected data and logs.
